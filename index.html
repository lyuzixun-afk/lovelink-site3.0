Welcome to LoveLink

This lightweight prototype shows core features you can demo offline: Registration/Login, User Catalog with filters, a simple Matching system, and a local “Live Chat”. Everything runs entirely in your browser using localStorage.

Course ready: Home • Register/Login • User Directory • Matching • Live Chat • Feedback & Reviews • About • Contact.

Data stays in your browser only for demo/school use. Clear via Settings → Clear site data.

Quick Start

  1. Click Create account and register.
  2. Open User Catalog to browse and filter users.
  3. Go to Matching, generate matches, and tap Like.
  4. When two users like each other (in demo, use two accounts), you can Chat.

Tip: Use your browser’s “New Private Window” to sign in as a second user for matching/chat demos.

About LoveLink

LoveLink helps people meet others who share similar interests, hobbies, and values, with a focus on safety and simple UX. This is a student-built prototype using Agile/Scrum ideas.

  • Better Matching via shared interests
  • Safety & Privacy – demo data stays in your browser
  • Simple UX and accessible design
  • Scalable (future) – ready to connect to real APIs
